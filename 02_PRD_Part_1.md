# 1. Product Overview

Discipline OS is an AI-first mobile application built with Flutter.

Unlike traditional productivity applications, Discipline OS combines

- Productivity
- AI
- Behavioral Science
- Health
- Analytics
- Native Android Features

into one ecosystem.

The application should become the user's daily operating system.
